利用Hexo+github搭建个人博客

By GH

发表于 11月 12 2014

文章目录

前文列举了搭建整个github个人主页所参考的链接和所有的必备知识，这篇博文主要针对一些细节地方出现的问题和解决办法简要叙述。

  1. 在前期注册好github账号，下载好node.js, msysgit后，在git bash 命令窗口中init好hexo，之后需要输入 npm install, 否则在hexo s通过本地查看网页时会出现乱码；

  2. 利用hexo d上传到github后，会有部分延迟才能看到更新的内容；

  3. 在安装多说评论系统时，一开始始终没找到在哪获取通用代码，最终在多说主页下体验一下的页面右下角找到了创建站点获取通用代码，点进去输入必要的信息后就能获取代码了。但是一开始上传到github仍就看不到评论框，在GoonX那注意到通用代码中还有三个地方需要自行替换

在他的博文中还记载了怎么把hexo右上角的搜索栏替换成百度搜索，说到这又开始觉得兲朝有些政策太恶心了，最近一段时间谷歌学术搜索论文只能通过师兄给的freegate翻墙弄，真是太麻烦了。。。

目前看来整个框架基本上都弄完了，剩下的事情就是把以前做的研究记录、笔记什么慢慢push上来。
